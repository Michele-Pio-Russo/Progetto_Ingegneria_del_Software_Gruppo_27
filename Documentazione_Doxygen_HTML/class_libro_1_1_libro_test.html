void Libro.LibroTest.testEquals ( )

Test del metodo equals.

Verifica la logica di uguaglianza tra due oggetti Libro, basata principalmente sul codice ISBN. Testa uguaglianza per riferimento, uguaglianza logica (stesso ISBN), disuguaglianza (ISBN diverso), confronto con null e confronto con oggetti di tipo diverso.

Precondizione
Due o più istanze di Libro (o null/altri oggetti) per il confronto.
Postcondizione
Il metodo equals deve restituire true solo se l'ISBN coincide, false altrimenti.

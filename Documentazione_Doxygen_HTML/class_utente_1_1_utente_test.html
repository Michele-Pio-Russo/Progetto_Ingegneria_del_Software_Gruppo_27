void Utente.UtenteTest.testEquals ( )

Test del metodo equals.

Verifica la logica di uguaglianza tra due oggetti Utente, basata principalmente sul nome. Testa uguaglianza per riferimento, uguaglianza logica (stesso nome), disuguaglianza (nome diverso), confronto con null e confronto con oggetti di tipo diverso.

Precondizione
Due o più istanze di Utente (o null/altri oggetti) per il confronto.
Postcondizione
Il metodo equals deve restituire true solo se il nome coincide, false altrimenti.
